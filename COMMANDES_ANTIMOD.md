# 📋 Guide Complet des Commandes Anti-Modération

## 🛡️ Protection Anti-Spam
**Commande**: `.antispam on/off`

Active ou désactive la protection contre les messages de spam du groupe.

**Fonctionnalités**:
- ✅ Détection automatique des messages spam
- ✅ Suppression des messages répétitifs
- ✅ Limitation du débit de messages
- ✅ Avertissement des utilisateurs

**Utilisation**:
```
.antispam on   - Active la protection
.antispam off  - Désactive la protection
```

## 📌 Prérequis

## ⚙️ Configuration Globale

Toutes ces protections peuvent être configurées individuellement pour chaque groupe. Le bot mémorise les préférences de chaque groupe indépendamment.

## 🔐 Sécurité

- Toutes les actions sont loggées
