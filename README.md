# 🎮 LUST BOT - WhatsApp Multi Device Bot

![LUST DEV](https://img.shields.io/badge/LUST-DEV-black?style=for-the-badge)

## ✨ Features

- 🛡️ **Group Management** - Contrôle total des groupes WhatsApp
- ⚡ **Commandes Puissantes** - Système de commandes avancé et personnalisable
- 🎯 **Fun Commands** - Commandes amusantes et interactives
- 🏗️ **Structure Propre** - Code organisé et maintenable
- 🚀 **Rapide & Silencieux** - Performance optimale et discrétion
- 📌 **Automatisme** - Gestion automatique des tâches
- 🔐 **Sécurisé** - Utilisation de Baileys pour la communication sécurisée

## 🛠️ Commandes

| Commande | Description |
|----------|-------------|
| `.menu` | Affiche le menu principal |
| `.help` | Aide et documentation |
| `.ping` | Vérifier la latence |
| `.kick` | Expulser un membre |
| `.tagall` | Mentionner tous les membres |

## 📦 Installation

```bash
npm install
node index.js
```

## 🚀 Utilisation

Lancez le bot avec la commande ci-dessus et scannez le code QR avec votre WhatsApp.
